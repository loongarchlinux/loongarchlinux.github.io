[Github Packages](https://github.com/orgs/loongarchlinux/packages) 提供了发布 Docker 镜像的功能，我们现在为 Loong Arch Linux 提供了 Docker 镜像。

您可以使用以下命令获取：

```
docker pull ghcr.io/loongarchlinux/archlinux:latest
```
